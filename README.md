# AI Knowledge Vault

Hi everyone! This is my beginner website project called **AI Knowledge Vault**. It is a modern, dark-themed landing page for a second-brain app. I updated it recently to add a cool AI futuristic design.

## 🚀 About The Project

Before, my project was simple. Now, I tried to make it look like a premium, modern AI website. I learned a lot about design, colors, and how to make things look clean.

This is just a frontend landing page, so there is no backend for now. But it looks really alive because of the CSS animations!

## ✨ Features

- **Modern Dark Theme:** It has a nice dark background with purple and blue glowing colors.
- **Glassmorphism:** The cards look like glass. It gives a very premium feel.
- **Floating Animations:** The images move slowly, which makes the website feel alive.
- **Responsive:** It looks good on computers and also adapts to smaller screens.
- **Icons:** I used FontAwesome icons instead of simple images to make it look professional.

## 🛠️ Built With

- HTML5
- CSS3 (Vanilla CSS, no frameworks here!)
- FontAwesome (for nice icons)
- AI Generated Images

## 💻 How To Run

* [ ] It is very easy to run! You don't need any special software.

1. Download or clone this project.
2. Open the folder.
3. Double-click on the `index.html` file.
4. It will open in your web browser.
